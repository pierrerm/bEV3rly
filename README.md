# bEV3rly
bEV3rly is a robot built with Lego Mindstorms EV3 hardware and programmed with the Java-based leJOS firmware.

![alt text](https://github.com/pierrerm/bEV3rly/blob/master/DPMPoster-1.png "Poster DPM")
